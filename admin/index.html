GitHub connection

Paste a GitHub personal access token to enable editing. Fine-grained, scoped to this repo only, with Contents: read & write.

How to create the token
  1. Go to github.com/settings/personal-access-tokens/new.
  2. Token name: lostlands-admin. Expiry: your call (90 days is sensible).
  3. Repository access → Only select repositories → choose mrob-dev/lostlands.
  4. Permissions → Repository permissions → Contents: Read and write.
  5. Generate, copy the github_pat_… string, paste it below.
